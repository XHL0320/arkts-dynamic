# Stack Non-standard Behavior

本目录用于看护当前 Error 类型下的具体语义触发场景。

## Scope

非标准 stack 行为

## Typical Scenarios

- stack 非 ECMA 标准强制属性
- ArkRuntime stack
- 引擎差异
- 不作为标准必测

## Watch Points

- stack 非 ECMA 标准强制属性
- ArkRuntime stack
- 引擎差异
- 不作为标准必测

## ArkTS Boundary

记录 ArkTS 静态语言、动态 ArkTS、ArkRuntime 的行为边界。若运行时错误在 ArkTS 静态语言中被前移为编译期错误，应在本目录覆盖矩阵和 issue 文件中记录。
