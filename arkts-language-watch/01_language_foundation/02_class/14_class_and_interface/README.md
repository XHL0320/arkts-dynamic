# Class and Interface Watch

本目录用于看护 ArkTS Class 类语言特性的聚合场景。

## Watch Points

- class implements interface
- class 缺少 interface 成员
- class 方法实现 interface 方法
- class 字段实现 interface 属性
- interface 继承 class 类型，如 ArkTS 支持
- abstract class 与 interface 对比
- private/protected 成员对 implements 的影响
- 结构检查还是名义检查
